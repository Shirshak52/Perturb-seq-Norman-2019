# CRISPR-MOD: End-to-End Perturb-seq Analysis of CEBPE-Driven Granulocytic Differentiation in K562 Cells

[![Python 3.13+](https://img.shields.io/badge/python-3.13+-3776ab.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**End-to-end single-cell Perturb-seq analysis pipeline** applied to the **Norman et al. 2019 K562 CRISPRa dataset**, focusing on **CEBPE overactivation**. Covers quality control, preprocessing, clustering, differential expression, and pathway enrichment analysis, demonstrating the complete computational biologist workflow for CRISPR screen data.

## Key Results

| Analysis                    | Finding                                                                          |
| :-------------------------- | :------------------------------------------------------------------------------- |
| DE genes (CEBPE vs Control) | 1,228 significant genes (604 up, 624 down)                                       |
| CEBPE fold change           | ~1,413x (confirms CRISPRa activation)                                            |
| Top upregulated pathways    | Neutrophil degranulation, Myeloid cell differentiation, Innate immune activation |
| Top downregulated pathways  | MYC targets, E2F targets, G2M checkpoint, Erythroid fate             |
| Cell cycle confounding      | 0.9% overlap with cell cycle genes; minimal                                     |

## Method Overview

### Pipeline Architecture
- **QC and Cleaning:** Scrublet doublet detection, mitochondrial gene filtering, count-based cell filtering
- **Preprocessing:** CP10K normalization → log1p transformation → 3,000 HVGs
- **Clustering:** PCA (50 components) → UMAP generation → Tirosh et al. 2016 cell cycle scoring
- **DE Analysis:** Wilcoxon rank-sum test, Benjamini-Hochberg correction, gene prefiltering (≥10% expression in either group)
- **Pathway Enrichment:** ORA via Enrichr on upregulated and downregulated gene lists separately

### Analytical Focus
Analysis is restricted to CEBPE perturbation vs Control — chosen for its known role as a master regulator of granulocytic differentiation, making it an ideal case study for validating the complete pipeline against established biology.

## Key Findings

### Differential Expression
- 1,228 DE genes identified (adj. p-value < 0.05, |ln fold change| > 0.5)
- CEBPE confirmed as top upregulated gene (~1,413x fold change) — positive control validation
- Top upregulated: PLEK, TRGC1, LST1, FYB — myeloid identity markers
- Top downregulated: GYPA, GYPB, HBG2, HEMGN — erythroid markers

### Pathway Enrichment
- **Upregulated:** Neutrophil degranulation, myeloid cell differentiation, innate immune effector functions, PU.1-CEBP regulatory axis reactivation
- **Downregulated:** MYC targets, E2F targets, G2M checkpoint (cell cycle exit), erythroid fate suppression (EKLF and GATA2 target genes), ribosome biogenesis suppression

### Biological Interpretation
CEBPE overactivation in K562 CML cells activates a granulocytic differentiation program, suppresses proliferation, and redirects cells away from their default erythroid fate — consistent with CEBPE's known role in normal hematopoiesis and demonstrating partial reversal of the CML proliferative phenotype.
